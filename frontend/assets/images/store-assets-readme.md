# Assets para Google Play Store - SinAdicciones

## Imágenes Requeridas

### 1. Ícono de la App (icon.png)
- **Tamaño**: 512x512 px (para Play Store) / 1024x1024 px (para iOS)
- **Formato**: PNG sin transparencia
- **Diseño sugerido**: 
  - Fondo verde (#10B981)
  - Símbolo de corazón o manos entrelazadas
  - Texto "SA" o símbolo de infinito (recuperación continua)

### 2. Ícono Adaptativo Android (adaptive-icon.png)
- **Tamaño**: 1024x1024 px
- **Formato**: PNG con transparencia
- **Solo el elemento del centro** (sin fondo)

### 3. Splash Screen (splash-icon.png)
- **Tamaño**: 1284x2778 px
- **Color de fondo**: #10B981 (verde esmeralda)
- **Logo centrado**

### 4. Gráfico de Funciones (feature-graphic.png)
- **Tamaño**: 1024x500 px
- **Usado en**: Cabecera de Play Store
- **Diseño sugerido**: Imagen inspiradora con texto "SinAdicciones - Tu camino hacia la recuperación"

### 5. Capturas de Pantalla (screenshots/)
- **Cantidad**: Mínimo 2, recomendado 4-6
- **Tamaño**: 1080x1920 px (teléfono)
- **Mostrar**: Dashboard, seguimiento, logros, perfil profesional

## Herramientas Recomendadas para Crear Assets

1. **Canva** (gratis): https://canva.com
   - Plantillas para iconos de apps
   - Fácil de usar

2. **Figma** (gratis): https://figma.com
   - Diseño profesional
   - Templates de app icons

3. **App Icon Generator**: https://appicon.co
   - Genera todos los tamaños automáticamente
